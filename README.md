# Desktop-Assistant
Basic Desktop Assistant using Speech Recognization, pyttsx3, Pyaudio, and other modules

Action: 
It takes your speak command by using Speech recognizer and processes and analyzes it using Pyaudio and gives a voice output using pyttsx3 module or does its expected work.

It is capable of doing ---  #Opening specific files and folders and drives  #Opening and searching in Google  #Opening and searching in youtube   #Playing Music and recordings
                      
                            #Opening and searching by name in facebook  #Opening my face detection webcam (done by me)   #Opening my snake game  #Searching emails  
                            
                            #Reading and speaking any text folder in the desktop  #Wikipedia search and telling short details  #Telling me information about time,date,month,day
                            
                            #Greeting me and any of my guests
